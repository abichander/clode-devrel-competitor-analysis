# Competitor Research (10 Companies)

## 1. Netlify
Website: https://www.netlify.com  
Founder(s): Mathias Biilmann  
Location: US  
Primary category: Platform Engineering / Internal Developer Platforms  

What they do:  
Platform for deploying, hosting, and managing modern web apps with CI/CD baked in.

Target user (ICP): Frontend & full-stack developers, startups  

Key features:
- CI/CD pipelines
- Serverless functions
- Edge deploys
- Git-based workflows

Pricing: https://www.netlify.com/pricing/  

Positioning: “Build, deploy, and scale modern web apps”  

Overlap with Clode:
- Deployment automation
- Infra abstraction

How Clode is different:
- Focuses on backend, infra automation, and SDLC ops

DevRel angle:
- Frontend DevOps vs full infra automation

Content hooks:
- “Frontend DevOps ≠ Platform Engineering”
- “Why deployment is only 30% of infra automation”

## 2. Platformatic
Website: https://docs.platformatic.dev  
Founder(s): Matteo Collina  
Location: EU  
Primary category: Platform Engineering / IDP  

What they do:  
Composable backend platform for building APIs and services.

Target user: Backend & platform engineers  

Key features:
- API-first backend
- Plugin system
- Observability support

Pricing: OSS / Enterprise  

Positioning: “The open source backend platform”  

Overlap with Clode:
- Platform tooling

How Clode is different:
- Automates infra + ops beyond backend services

DevRel angle:
- Backend frameworks vs full lifecycle automation

Content hooks:
- “Backend platforms still need infra automation”
- “IDPs aren’t just APIs”
